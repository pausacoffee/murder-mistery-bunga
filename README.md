# 룰북 대체 웹앱

간단한 정적 웹앱입니다. 설치 없이 바로 실행할 수 있습니다.

## 프로젝트 구성
- `index.html`: 슬라이드(7장) 구조
- `styles.css`: PPT 비율(16:9) 화면 스타일
- `app.js`: 페이지 이동, 타이머 기능

## 실행 방법
1. 파일 탐색기에서 `D:\Project\WebApp\01_bunga\index.html`을 더블클릭합니다.
2. 또는 VS Code Live Server 등으로 실행해도 됩니다.

## 현재 구현된 기능
- PPT 기본 비율(16:9)
- 우상단 `이전 / 다음` 버튼
- 키보드 좌우 방향키 이동
- 모바일 터치 스와이프 이동
- 타이머 기능
  - 시작/일시정지 버튼
  - +1분, -1분
  - 50초 이하에서 감소 버튼 클릭 시 `타이머 가 종료 됩니다` 확인창
  - 확인 시 즉시 종료 및 `-종료-` 표시
- 1~7장 기본 내용 반영
- 3장/6장 스크롤 가능한 텍스트 영역
