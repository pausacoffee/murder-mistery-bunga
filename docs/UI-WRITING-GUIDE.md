# 문구/표기 가이드

## 1. 표기 원칙
- `Part` -> `파트`
- `Round` -> `라운드`
- 표기 형식: `라운드 1` (숫자 뒤치기 `1라운드` 사용 금지)

## 2. 슬라이드 타이틀 규칙
- 장 번호 접두(`1장:`, `2장:` 등) 사용하지 않음
- 파트 슬라이드는 `[파트 n] ...` 형식 사용
- 예: `[파트 2] 라운드 3: 탐색과 정보 공유`

## 3. 버튼 문구 규칙
- 파트 종료 버튼은 행동 중심 문구 사용
  - `이야기 카드 A 공개`
  - `이야기 카드 B 공개`

## 4. 확인 모달 문구 규칙
- 질문 + 다음 행동을 명확히 구분
- 강조 규칙
  - 핵심 오브젝트: `confirm-primary-line` (볼드)
  - 다음 진입 파트: `confirm-warning-line` (붉은 볼드)

현재 문구:
- 파트 2 진입: `이야기 카드 A가 공개되었습니까? / 파트 2로 진입합니다.`
- 파트 3 진입: `이야기 카드 B가 공개되었습니까? / 파트 3로 진입합니다.`

## 5. 새 페이지 추가 시 체크리스트
1. `index.html`에 `data-index`, `data-title` 추가
2. 타이머 페이지면 `data-part-info-title`/`data-part-info-topic` 정의
3. 규칙 팝업이 필요하면 `app.js > RULE_CONTENT_MAP` 키 추가
4. 문구가 가이드(파트/라운드 표기)에 맞는지 검수
