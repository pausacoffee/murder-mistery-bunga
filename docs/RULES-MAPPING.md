# 규칙 팝업 매핑

## 공통 규칙 탭 (6페이지 + 책 아이콘)
- `능력과 파워`
- `승점`
- `전체`

## 파트 1/2/3 버튼-팝업 매핑
- `data-rule-topic` 값이 `RULE_CONTENT_MAP` 키와 일치하면 팝업이 열립니다.

### 파트 1 진행 순서 / i 버튼
- `행선지 상의 및 결정`
- `탐색 및 정보 공유`
- `인물 관찰 및 정보 공유`
- `다음 라운드 리더 지명`

### 파트 2 진행 순서 / i 버튼
- `행선지 상의 및 결정(3분)`
- `탐색과 정보 공유(15분)`
- `다음 라운드 리더 지명(1분)`

### 파트 3 준비 다이어그램
- `전체 회의 (10분)`
- `추리 페이즈 (각 플레이어당 1분)`
- `투표`
- `사제 선발`
- `엔딩` 노드는 팝업 미연결

## 팝업 콘텐츠 수정 위치
- 파일: `app.js`
- 객체: `RULE_CONTENT_MAP`
- 규칙 본문은 HTML 문자열로 관리
