# GM 운영 가이드

## 1. 진행 전 체크
- 화면 비율: 16:9 (TV/빔 권장)
- 조작 위치: GM만 화면 근처에서 조작
- 플레이어 안내: 핵심은 "현재 라운드/현재 페이즈/남은 시간"

## 2. 파트별 운영 흐름
1. 파트 1
- 인덱스 7: `[파트 1] 진행 순서`
- 인덱스 8~19: 라운드 1~3 (행선지 상의/탐색/인물 관찰/리더 지명)
- 헤더 버튼: `이야기 카드 A 공개`

2. 파트 2
- 인덱스 20~23: 스토리/준비/설정서 확인/진행 순서
- 인덱스 24~35: 라운드 1~4 (행선지 상의/탐색/리더 지명)
- 헤더 버튼: `이야기 카드 B 공개`

3. 파트 3
- 인덱스 36: `[파트 3] 스토리`
- 인덱스 37: `[파트 3] 준비` (세로 다이어그램)
- 인덱스 38~42: 전체 회의/추리/투표/사제 선발/엔딩

## 3. 조작 포인트
- `i` 버튼: 현재 페이즈 규칙 팝업 오픈
- 책 아이콘: 탭형 규칙 팝업 오픈
- 마지막 라운드에서 다음 클릭 시 파트 종료 확인 모달 자동 표시

## 4. 타이머 운영 팁
- 타이머가 1분 이하일 때 `-1` 클릭 시 종료 확인 모달 노출
- 페이즈 전환 시 기존 타이머는 자동 일시정지
- 탭 비활성화(브라우저 숨김) 시 타이머 자동 일시정지

## 5. 파트 3 다이어그램 노드
- 1: 전체 회의 (10분)
- 2: 추리 페이즈 (각 플레이어당 1분)
- 3: 투표
- 4: 사제 선발
- 5: 엔딩 (규칙 팝업 없음)
